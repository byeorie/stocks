📈 주식 대시보드 설정

Google Apps Script 웹앱 URL을 연결하면 구글 시트 데이터를 실시간으로 가져옵니다.

1
구글 시트 → 확장 프로그램 > Apps Script
2
stock_apps_script.gs 코드 붙여넣기 후 저장
3
setup() 함수 실행 → 시트 자동 생성
4
배포 > 새 배포 > 웹앱 → 액세스: 모든 사용자 → URL 복사
